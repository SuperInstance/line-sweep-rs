# line-sweep-rs

Computational geometry via line sweep in pure Rust.

## Features

- **Line Segment Intersection** — Bentley-Ottmann algorithm
- **Closest Pair of Points** — O(n log n) divide-and-conquer
- **Rectangle Intersection** — Sweep-based overlap detection

No external dependencies. Pure `std` Rust.
